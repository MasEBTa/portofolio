Overview Tentang Github

Cashere App

Ide dasar proyek ini adalah sebuah aplikasi Ponit Of Sales (kasir).

Detail Aplikasi

  1. Aplikasi Spa
  2. Pengguna dapat menambahkan barang dagang, supplier, dan menambah stock.
  3. Pengguna dapat melakuan penjualan.
  4. Hasil penjualan dapat dicetak.
  5. Responsive/mobile friendly.

Teknologi

Saya membuat halaman ini menggunakan Laravel dan vue js
