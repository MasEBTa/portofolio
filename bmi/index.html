Kalkulator BMI

Ide dasar proyek ini adalah sebuah aplikasi berbasis website yang digunakan untuk mengukur Masa Index Tubuh. Hasil dari pengukuran tersebut akan menghasilkan 4 area :

  1. Sangat kurus
  2. Kurus
  3. Ideal
  4. Kegemukan
  5. Over Dosis
Hasil pengukuran tersebut akan memberikan feedback akan pengingat untuk menjaga kesehatan tubuh.

Detail Aplikasi

  1. Pengguna tinggal memasukkan data yang sesuai.
  2. Pengguna menekan tombol submit.
  3. Pengguna website akan mengembalikan feedback sesuai data.

Teknologi

Saya membuat halaman ini menggunakan Html dan Php.
