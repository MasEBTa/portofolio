Instagram Login Page Clone

Ide dasar proyek ini adalah cloning dari halaman login Instagram Website.

Detail Aplikasi

  1. Hanya 1 halaman. Saya buat semirip mungkin dengan halaman login Instagram.
  2. Menampilkan slider gambar (Carousel) di sebelah kiri yang bergantian 3 gambar setiap 2 detik (2000ms)
    • Screen Shoot Chat
    • Screen Shoot Story Instagram
    • Screen Shoot Home Page
  3. Menampilkan hover label pada input (username/password) ketika mengetikkan suatu karakter.
  4. Fitur hide/show pada input password.
  5. Responsive/mobile friendly.

Teknologi

Saya membuat halaman ini menggunakan Html, Css, Javascript.

Latar Belakang Pembuatan

Halaman website ini saya buat untuk mengukur kemampuan saya pada Pembuatan design website. Ada beberapa hal baru yang saya pelajari pada pembuatan halaman ini seperti :
  • Pembuatan Carousel dan penggunaan setTimeout pada Javascript.
  • Pemahaman penggunaan Flex pada css yang sangat powerfull untuk design tampilan website.
  • Pengaturan tata letak pada sebuah halaman website.
