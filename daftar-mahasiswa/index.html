Pengelola Daftar Mahasiswa

Ide dasar proyek ini adalah sebuah aplikasi berbasis website yang digunakan untuk mengelola daftar mahasiswa.

Detail Aplikasi

Fokus utama dalam pembuatan proyek ini adalah mengenai sistem arsitektur MVC (Model-View-Controller) yang memisahkan antara logika aplikasi, View atau penampil halaman, dan controller router.

Untuk mengatur controller saya menggunakan .httaccess sebagai Controller url yang di berikan.

Teknologi

Saya membuat halaman ini menggunakan Html, Css, Bootstrap, Javascript dan Php.

Latar Belakang Pembuatan

Aplikasi ini saya buat ketika saya akan mempelajari mengenai Framework berbasis php. Karena sebagian besar framework menggunakan arsitektur yang memisahkan antara logika dan view aplikasi. Setelah pembuatan aplikasi ini saya dapat lebih memahami secara garis besar proses arsitektur dari sebuah aplikasi yang utuh.
