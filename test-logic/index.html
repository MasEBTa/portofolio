Test Logika Pemroggraman

Ide dasar proyek ini adalah mengerjakan beberapa test logika proggramming namun saya buat layaknya sebuah kertas exam dengan jawaban yang interaktif.

Detail Aplikasi

  1. Terdapat 10 soal test logika.
  2. Jawaban dari test logika tersebuat menggunakan bahasa Javascript.
  3. Jawaban dari test logika tersebuat tidak tampil di halaman, melainkan hanya hasil output yang diharapkan dari tiap-tiap soal.

Teknologi

Saya membuat halaman ini menggunakan Html, Css, Javascript.

Latar Belakang Pembuatan

Halaman website ini saya buat sebagai salah satu tugas dari kelas laravelXvue yang disediakan oleh Eduwork. Tujuan dari tugas ini adalah mengukur kemampuan analisa dan logika terutama pada pengguna bahasa JavaScript.

Demo Aplikasi
