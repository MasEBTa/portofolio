Luas Bangun Datar

Ide dasar proyek ini adalah membuat kumpulan rumus-rumus luas bangun datar sederhana.

Detail Aplikasi

  1. Saat ini ada sekitar 5 luas bangun datar.
  2. User tinggal memasukkan angka dasar perhitungan luas dari tiap tiap bangun, lalu akan keluar hasil beserta gambaran bangun datar tersebut.
  3. Pemrosesan perhitungan dilakukan di dalam halaman menggunakan Php.

Harapan

  1. Akan ada update rumus-rumus lain kedepannya
  2. Akan ditampilkan dalam tampilan yang lebih baik kedepannya

Teknologi

Saya membuat halaman ini menggunakan Html, Css, Javascript, dan Php.

Latar Belakang Pembuatan

Halaman website ini saya buat untuk lebih memahami proses server side proggraming berjalan terutaman dalam bahasa pemrogramman Php. Karena itu saya hanya fokus dalam penggunaan Php dalam projek ini.
