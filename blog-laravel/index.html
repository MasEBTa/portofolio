Blog Laravel

Ide dasar proyek ini adalah sebuah Blog menggunakan bahasa laravel.

Detail Aplikasi

Fokus utama dalam pembuatan proyek ini adalah crud laravel dan upload gambar menggunakan storage:link.

Teknologi

Saya membuat halaman ini menggunakan Html, Css, Bootstrap, Javascript, Php, dan Laravel.

Latar Belakang Pembuatan

Aplikasi ini saya buat ketika saya mulai mempelajari Laravel framework. dapat dikatakan bahwa ini adalah proyek pertama saya dalam membuat sebuah aplikasi yang utuh. Saya mempelajari berbagai hal baru dari pembuatan halaman ini seperti :

  1. Pengguna storage:link yang saya gunakan untuk upload gambar setika membuat postingan baru, maupun mengedit postingan.
  2. Penggunaa middleware untuk proses autorisasi.
  3. Memahmi proses MVC menggunakan framework laravel.
  4. dan lain sebagainya
