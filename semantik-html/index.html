Eduwork Landing Page Clone

Ide dasar proyek ini adalah cloning dari halaman landing Page eduwork.id.

Detail Aplikasi

    Hanya 1 halaman. yang terdiri dari :
  1. Header
  2. Jumbotron
  3. Card feature.
  4. Responsive/mobile friendly.

Teknologi

Saya membuat halaman ini menggunakan Html, Css, Bootstrap.

Latar Belakang Pembuatan

Halaman website ini saya buat sebagai salah satu tugas dari kelas laravelXvue yang disediakan oleh Eduwork. Tujuan dari tugas ini adalah mengukur kemampuan dalam pembuatan tampilan website menggunakan bootstrap sebagai framework CSS, HTML, dan JavaScript untuk mempermudah pembuatan website responsif dan mobile-friendly.
